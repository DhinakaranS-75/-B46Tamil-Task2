Question-1:
  Write a blog on the Differrence between document & window objects
      Answer:
        Document:
          (1).It represents the document loaded inside the window or browser.
          (2).It is the root element of the document object model.
          (3).It is an object of window.
          (4).document.title will return the title of the document
          (5).logically:
              document:{ properties}

        Window:
          (1).It represents the browser window in which you are seeing the content.
          (2).The window is the global element for all objects, functions, etc.
          (3).It is an object of the browser.
          (4).window.document.title will return the title of the document
          (5).logically:
              window:{
                  document:{properties}
              }
